Answers to the two questions:

The course number and name of the course are: 13FL CIT-1157-C02 HTML and Web Publishing.

The format for the img tag is: <img [src="<relative/absolute reference>" alt="alternate text to describe the image for blind users(with screen readers) etc."]/>
FOR EXAMPLE:
  <img src="../images/W3C_Icon.svg" alt="W3C LOGO" width="250" height="250" />
  <img src="../images/happy.jpg" alt="a happy face" />
